日期处理函数

​ 在 Mysql数据库中，已经给我们内定了许多的函数，如下使用所示：

获取系统当前日期时间： SYSDATE()、NOW()

获取系统当前日期：CURDATE()

获取系统当前时间：CURTIME()

获取给定日期的年： YEAR(date)，求系统当前日期的年份

获取给定日期的季度： QUARTER(date)，求系统当前日期的季度

获取给定日期的月：MONTH(date)，求系统当前日期的月份

获取给定日期是星期几：DAYOFWEEK(date)，求系统当前日期的是星期几

1=星期天，2=星期一，…………，7=星期六

获取该日期为当月的第几天：DAYOFMONTH(date)，求系统当前日期的是这个月的第几天

获取该日期为当年的第几天：DAYOFYEAR(date)，求系统当前日期的是今年的第几天

操作时间，为指定日期增加一个时间间隔的函数：DATE_ADD(date，INTERVAL expr unit)

获取两个日期间的时间差：TIMESTAMPDIFF(求时间的什么，begindate，enddate)
