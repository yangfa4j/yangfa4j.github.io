组合模式

​ 组合模式解决这样的问题：

​ 当我们要处理的对象可以生成一颗树形结构，而我们要对树上的结点和叶子进行操作时，它能够提供一致的方式，而不用考虑它是结点还是叶子。

​ 这个系统中所有对象都具有相同的方法。他们都继承共同的抽象类。
