Ribbon 客户端的负载均衡

将客户端的请求平均的分配多个服务器上，保证合理的资源调用。

Ribbon是一个基于 HTTP 和 TCP 的客户端负载均衡器，当我们将 Ribbon 和 Eureka 一起使用时，Ribbon 会从Eureka注册中心去获取服务端列表，然后进行轮询访问以到达负载均衡的作用。

服务端是否在线这些问题则交由Eureka去维护。
